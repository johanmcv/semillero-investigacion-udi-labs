Este ejercicio práctico tiene como objetivo aprender los conceptos básicos del desarrollo de aplicaciones web usando el stack TALL (Tailwind, AlpineJS, Laravel, Livewire) y Filament PHP.

En este ejercicio se planea:

- Configurar un proyecto Laravel 11 y conectar con MySQL.
- Instalar y configurar Filament PHP para administración de recursos.
- Crear un CRUD básico de usuarios con acciones de creación, edición y eliminación.
-Implementar roles y permisos mediante Filament Shield.
-Probar el flujo de administración y validar que los permisos funcionen correctamente.

Objetivo: familiarizarse con Laravel y Filament, comprender la creación de recursos CRUD, gestión de roles y permisos, y sentar las bases para el desarrollo de aplicaciones tipo SaaS usando el stack TALL.
